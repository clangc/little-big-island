# The Little Big Island — marketing site

Static site. `index.html` = founding-family landing page. `island-finder/` = the free quiz.

## Deploy (Vercel)
1. Import this repo at vercel.com/new — framework preset: "Other", no build command, output = root.
2. Add domain thelittlebigisland.com in Vercel → Project → Settings → Domains, then set your registrar's DNS as Vercel instructs.

## Before launch
- In index.html: add the founder photo (search `founder-photo`) and paste the Stripe payment link (search `PAYMENT_LINK_URL`).
